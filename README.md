# AI Notes Summarizer + Quiz Generator

A beginner-friendly Streamlit website where students can paste notes and generate:
- Short/medium/detailed summaries
- MCQ-style quiz questions

## How to run

1. Install Python.
2. Open terminal in this folder.
3. Run:

```bash
pip install -r requirements.txt
streamlit run app.py
```



## Note

This first version uses simple Python logic. Later, you can connect it to an AI API for better summaries and quizzes.
